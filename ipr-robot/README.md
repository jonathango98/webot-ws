# IPR Robot Project

This project focuses on controlling an IPR robot using a camera.

## Project Setup

This project is a Webots simulation for controlling an IPR robot. The robot is controlled using a Python script that uses `ikpy` for inverse kinematics and `OpenCV` for camera processing.

## Installation

1.  **Install Webots:** Make sure you have Webots installed. You can download it from [https://cyberbotics.com/](https://cyberbotics.com/).
2.  **Install Python dependencies:** The required Python libraries are listed in the `requirements.txt` file. You can install them using pip:

    ```bash
    pip install -r requirements.txt
    ```
